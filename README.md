# Gemini Code Assistant Korean - AiCrafter

Gemini Code Assistant Korean helps you use the Gemini CLI more efficiently with optimized Korean input support. This extension automatically launches the Gemini CLI and provides a convenient input panel specifically designed for Korean developers.

## ✨ Key Features

- 🇰🇷 **Korean Input Optimization**: IME composition detection prevents premature sending during Korean typing
- ⚡ **Quick Send**: Send commands with Ctrl + Enter shortcut
- 🔄 **Auto-flush Option**: Automatically clear input after sending (toggleable)
- 🎨 **Intuitive UI**: Color-coded buttons for better user experience
- 📁 **File/Folder Integration**: Quick path addition with context awareness
- 🚀 **Turbo Mode**: Fast development workflow

## 🚨 Important

**To use Gemini Code Assistant Korean, you must first have the Gemini CLI installed and running.**

This extension is specifically optimized for Korean developers and Windows/macOS environments.

## 🚀 Usage

1. **Install and Open**: Install the extension and open VS Code in your project directory
2. **Panel Appears**: The **Gemini Code Assistant Korean** panel will automatically appear in the Explorer tab
3. **Start Gemini**: Click the **"Gemini"** button to automatically launch the Gemini CLI
4. **Type Message**: Enter your command or question in the text input area
5. **Send Message**: Press **Ctrl + Enter** to send to the Gemini CLI
6. **Auto-flush Option**: Check **"Auto-flush message after send"** to automatically clear the input field after sending
7. **Turbo Mode**: Click **"Turbo"** button to run Gemini with full permissions for advanced operations

## 🎯 Features

- **"Send" shortcut**: Send commands to the Gemini CLI (Shortcut: Ctrl + Enter)
- **"Flush" button**: Clear the input area instantly  
- **"Gemini" button**: Automatically launch a new Gemini CLI session
- **"Add File" button**: Add current file path to input for context
- **"Add Folder" button**: Add current folder path to input for context
- **"Turbo" button**: Launch Gemini with full permissions for advanced file operations
- **Auto-flush checkbox**: When checked, automatically clears input field after sending message
- **Korean indicator**: Blue dot appears when Korean IME is composing characters
- **Character counter**: Real-time display of input length

# Gemini Code Assistant Korean - AiCrafter (한국어)

Gemini Code Assistant Korean은 한국어 입력에 최적화된 Gemini 어시스턴트입니다. VS Code에서 Gemini CLI가 자동으로 실행되며, 한국 개발자를 위해 특별히 설계된 편리한 입력 패널을 제공합니다.

## ✨ 주요 기능

- 🇰🇷 **한국어 입력 최적화**: IME 조합 감지로 한국어 타이핑 중 잘못된 전송 방지
- ⚡ **빠른 전송**: Ctrl + Enter 단축키로 즉시 전송
- 🔄 **자동 플러시 옵션**: 전송 후 입력창 자동 초기화 (선택 가능)
- 🎨 **직관적 UI**: 색상별 버튼으로 향상된 사용자 경험
- 📁 **파일/폴더 연동**: 컨텍스트 인식 경로 추가 기능
- 🚀 **터보 모드**: 빠른 개발 워크플로우

## 🚨 중요사항

**Gemini Code Assistant Korean을 사용하려면 먼저 Gemini CLI가 설치 및 실행되어 있어야 합니다.**

이 확장 프로그램은 한국 개발자와 Windows/macOS 환경에 특별히 최적화되었습니다.

## 🚀 사용법

1. **설치 및 열기**: 확장 프로그램을 설치하고 프로젝트 디렉토리에서 VS Code를 엽니다
2. **패널 표시**: Explorer 탭에서 **Gemini Code Assistant Korean** 패널이 자동으로 나타납니다
3. **Gemini 시작**: **"Gemini"** 버튼을 클릭하면 Gemini CLI가 자동으로 실행됩니다
4. **메시지 입력**: 텍스트 입력 영역에 명령어나 질문을 입력합니다
5. **메시지 전송**: **Ctrl + Enter**를 눌러 Gemini CLI로 전송합니다
6. **자동 삭제 옵션**: **"전송 후 메시지 자동삭제"**를 체크하면 메시지 전송 후 입력창이 자동으로 초기화됩니다
7. **터보 모드**: **"Turbo"** 버튼을 클릭하면 모든 권한이 부여된 Gemini가 실행되어 고급 작업을 수행할 수 있습니다

## 🎯 주요 기능

- **"전송" 단축키**: Gemini CLI에 명령어 전송 (단축키: Ctrl + Enter)
- **"Flush" 버튼**: 입력 영역 즉시 초기화
- **"Gemini" 버튼**: Gemini CLI 세션을 자동으로 시작
- **"Add File" 버튼**: 현재 파일 경로를 컨텍스트로 입력에 추가
- **"Add Folder" 버튼**: 현재 폴더 경로를 컨텍스트로 입력에 추가
- **"Turbo" 버튼**: 모든 권한이 부여된 Gemini 실행으로 고급 파일 작업 수행
- **자동 삭제 체크박스**: 체크 시 메시지 전송 후 입력창 자동 초기화
- **한국어 인디케이터**: 한국어 IME 문자 조합 중 파란 점 표시
- **글자수 카운터**: 실시간 입력 길이 표시
